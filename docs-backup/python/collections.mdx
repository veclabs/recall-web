# Python SDK - Collections

## Create a collection

```python
collection = sv.collection(
    "my-collection",
    dimensions=1536,
    metric="cosine"    # "cosine" | "euclidean" | "dot_product"
)
```

## Collection stats

```python
stats = collection.stats()
print(f"Vectors: {stats.vector_count}")
print(f"Dimensions: {stats.dimensions}")
```
