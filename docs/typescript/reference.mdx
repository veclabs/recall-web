# TypeScript API Reference

## SolVec

```typescript
new SolVec(options: SolVecConfig)
```

```typescript
interface SolVecConfig {
  network: 'mainnet-beta' | 'devnet' | 'localnet'
  walletPath?: string      // path to Solana keypair JSON
  rpcUrl?: string          // custom RPC endpoint
  shadowDrive?: boolean    // Phase 5: enable Shadow Drive (requires walletPath)
}
```

## sv.collection(name)

Returns a `Collection` instance.

```typescript
const index = sv.collection("my-index");
```

## index.upsert(vectors)

Upserts vectors into the collection.

```typescript
await index.upsert([
  {
    id: string,           // unique identifier
    values: number[],     // vector values
    metadata?: object     // optional metadata
  }
])
```

## index.query(options)

Queries the collection for nearest neighbors.

```typescript
const results = await index.query({
  vector: number[],   // query vector
  topK: number,       // number of results
  filter?: object     // optional metadata filter
})
```

Returns: `Array<{ id: string, score: number, metadata?: object }>`

## index.verify()

Posts a Merkle root to Solana and returns the proof.

```typescript
const proof = await index.verify();
// proof.merkleRoot      - 32-byte hex string
// proof.txSignature     - Solana transaction signature
// proof.solanaExplorerUrl - Explorer URL
```

## index.restoreFromShadowDrive()

```typescript
// Phase 5 — restore from Shadow Drive on cold start
async restoreFromShadowDrive(): Promise<boolean>
// Returns true if vectors were restored, false if nothing found
```
