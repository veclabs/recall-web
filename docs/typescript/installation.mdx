# Installation
```bash
npm install @veclabs/solvec
```

**Requirements:**
- Node.js 18 or higher
- TypeScript 5.0 or higher (if using TypeScript)

## Verify installation
```bash
node -e "const { SolVec } = require('@veclabs/solvec'); console.log('ok')"
```
