# TypeScript SDK - Client

## SolVec class

```typescript
import { SolVec } from "@veclabs/solvec";

const sv = new SolVec({
  network: 'devnet',           // 'devnet' | 'mainnet-beta' | 'localnet'
  walletPath: '/path/to/keypair.json',  // required for on-chain features
  shadowDrive: true,           // Phase 5: enable decentralized storage
  rpcUrl: 'https://...',       // optional custom RPC
});
```

## Environment variables

```typescript
const sv = new SolVec({
  network: "mainnet-beta",
  walletKey: JSON.parse(process.env.VECLABS_WALLET_KEY!),
});
```

<Note>
  `shadowDrive: true` requires `walletPath`. If walletPath is missing,
  Shadow Drive is disabled with a warning and all other operations continue normally.
</Note>
