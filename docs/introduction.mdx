# VecLabs

VecLabs is a vector database built for AI agents that need to prove what they remember.

Every write posts a 32-byte SHA-256 Merkle root to Solana. Immutable. Permanent. 
Verifiable by anyone. The query engine is Rust HNSW — 4.3ms p99 at 100K vectors.

## Why VecLabs

<CardGroup cols={2}>
  <Card title="4.3ms p99" icon="bolt">
    Rust HNSW core. No garbage collector. No JVM pauses. 
    No Python GIL. Vectors stay in memory.
  </Card>
  <Card title="On-chain proof" icon="shield-check">
    Every upsert posts a Merkle root to Solana. 
    $0.00025 per transaction. 400ms finality.
  </Card>
  <Card title="Your data" icon="lock">
    AES-256-GCM encrypted with a key derived from your 
    Solana wallet. VecLabs cannot read your data.
  </Card>
  <Card title="Pinecone compatible" icon="arrows-rotate">
    3 line changes to migrate. Same upsert/query API shape. 
    Plus index.verify() — Pinecone has no equivalent.
  </Card>
</CardGroup>
